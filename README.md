The project is split into four semesters, each one tackling a different step in the research
  - The first semester was dedicated for the documentation phase, getting familiar with the subjecte of the theme while also underlining the advantages that where brought with the introduction of robotic solutions
  - In the second and third semester we revisit the topics discussed in the previous one, reviewing them and making some necessary adjustments. We prioritise the most important functions of the robots and start to implement them on a theoretical level.
    A rough 3D model of the robot is also created during this semester
  - The fourth semester is reserved for improving existing solutions for the robot, including a partial overhaul of the 3D model.
